初めてのセメスターも終わり、溜まっていた読みたい本や見たい映画・ドラマ、プレイしたいゲームとなんだかんだ娯楽で忙しく過ごしていました:palm_tree:そんな休みもあっという間で来週から次の授業が始まります。今回は次受ける授業の科目と授業料についてまとめていきたいと思います:metal:

2021年の4月から9月のセメスターではComputational MathematicsとIntroduction to  Programming Ⅱを受けることにしました。そして同時進行でGoogle IT SupportをCourseraで受講してHow Computers Workの科目を免除する予定です。

#Computational Mathematicsについて
この科目ではNumber BaseやSequence,Trigonometry,Functions,Combinatoricsあたりをやってくみたいです。前回受けたDiscrete Mathematicsと被る部分もありますがどちらかというと基礎に近い感じです、DMより先にこちらを受けた方が良かったのではという気持ちがありますが、CMとDMは授業を開始する時期が決まっており、必然的に4月入学はCMから、9月入学はDMからというのが特徴です。ちなみにPBA(Performance Based Admission)といわれる、スキルや職歴で選考された生徒は最初にITP1とDM又はCMで一定の成績を納めないと他の科目を受けることはできません。

#Introduction to ProgrammingⅡに関して
前回のITPⅠではjsを使ったCreative CodingとGame Projectでしたが、ITPⅡでもjsを使ったCreative Codingなのは変わりないようです。ただ、以前受けた生徒の制作物を見てたらwebアプリを作っていたので結構実践的で楽しみです。

#Google IT Support(Coursera)に関して
How Computers Workの科目が免除されるので受け始めたのですが、内容もわかりやすく、なんとなくで理解してたハードウェアの知識もしっかり身に付くし、期間もそんなに長くなくて良いコースです。これは今週終了できそうなので大学の勉強と被らずに済みそうです。

#授業料について
授業料は登録する際に選んだ科目分を都度支払っていくシステムで今回は2科目で1,230£でした。あと今年度から日本で授業を受けている方は日本の消費税が掛かるよという連絡が来てましたので次回以降10%の消費税も掛かってくるようです。それでも普通に正規でイギリスの大学に留学生として行く場合に比べたらだいぶ安い方ではあるのですが、前からいる生徒さんによると授業料は年々増えてる傾向にあるそうなのでセメスターによって料金が変わってくる可能性があります。
個人的にコロナの影響で普通の大学もオンラインで学ぶことが当たり前になってきている中、今後オンラインで学ぶプラットフォームの需要は高くなってくるような気がしていて、その分料金も上がるのかなと思っています:money_with_wings:

#次の授業への準備
次も2科目なのであまり不安はないですが、前回のセメスターでの勉強方法があまり効率的ではなかったので何か参考になる本はないかなと思って見つけた本が良かったのでおすすめしておきます。

この本では最初に一般的に用いられる勉強法として出てくる「テキストの再読」や「集中練習」は極めて非効率だと書かれてます。また人は知っている、できるという錯覚にも陥りやすく、間違える事の重要性や知的能力を作る要素は自分次第で拡張できるとも言っています。前回noteで紹介したCourseraのLearning how to learnと似た内容にはなりますが、脳科学というよりも具体例を出した勉強法を説明していく内容なので今の自分の勉強法の見直しが出来るおすすめの本です。

#最後に
なんだかんだ日本の大学では起こらないようなハプニングに度々遭遇していて(授業料支払えない問題など)、その度に同じ生徒に助けてもらったりするので人は一人では生きていけないを実感しています。まこれも経験だと思いつつ丁度大学側からアンケートも来たので思いの丈をぶちまけたいと思います:bomb:
